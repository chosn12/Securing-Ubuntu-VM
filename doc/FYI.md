## Ubuntu Security Hardening ##

***This is basic security hardening to suit my testing environment. Feel free to secure your environment to your own personal specifications.***   
  - Remember, when editing the *sshd_config* file make a copy of the original and name the copy an easily identifiable name and edit original file, not the copy.
  - Test fail2ban to ensure your configuration is working as desired.
    - While testing check **`/var/log/auth.log`** to view real-time results.
